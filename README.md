# JobPulse 💼📈

A premium, interactive, and responsive job application tracker designed for job seekers to manage their application funnel (Kanban board + List views) with analytics and search/filter capabilities.

## ✨ Features
- **Funnel Management (Kanban Board)**: Drag and drop cards to change application status.
- **Detailed Table (List View)**: Toggle layouts to view applications in a spacious, searchable grid.
- **Dynamic Metrics Dashboard**: Tracks total applications, active applications, scheduled interviews, and interview success rate.
- **Smart Filtering & Sorting**: Filter by status and location types (Remote, Hybrid, On-site); sort by newest/oldest applied date, salary, or company name.
- **Data Portability**: Backup and restore data with JSON Import/Export.
- **Persistent Storage**: Utilizes `localStorage` for local client-side persistence without external database dependencies.
- **Premium Dark UI**: Built with glassmorphic cards, smooth animations, and custom SVGs.

## 🛠️ Technology Stack
- Vanilla HTML5
- CSS3 (Vanilla design variables, Flexbox & CSS Grid, Glassmorphic effects)
- Vanilla JavaScript (State controller, Drag-and-Drop, HTML5 Storage APIs)

## 🚀 Running Locally
Simply open `index.html` in any modern web browser or serve it locally:
```bash
# Example with Python
python -m http.server 8000
```
Open [http://localhost:8000/index.html](http://localhost:8000/index.html) in your browser.
